# ChessAgregator
Обзор
=====================
Мобильное приложение под IOS, которое помогает в организации и участии шахматных турниров. Вся удобная информация о турнире будет находиться у вас под рукой
### Архитектура:
* VIPER
### Основной стек технологий (в этом репозитории):

* Swift 5.4
* UIKit 
* Firebase
